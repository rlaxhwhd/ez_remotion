2 영상 / 이미지 / 오디오 업로드

왼쪽 패널 → "⬆ 파일 업로드" 버튼 클릭 mp4, webm, mov, avi, mkv / png, jpg, gif, webp / mp3, wav, aac, m4a 등 지원
파일 선택 (여러 파일 한 번에 가능) 영상: 플레이헤드 위치에 클립으로 추가됨 · 첫 영상의 해상도가 캔버스 크기로 자동 적용됨
타임라인과 클립 목록에 즉시 반영 왼쪽 하단 "클립 목록"에서 클릭하면 선택 · 오른쪽 패널에서 속성 편집 가능

텍스트 / 도형 추가

  • ＋ 텍스트 — "여기에 텍스트" 클립이 플레이헤드 위치에 추가. 오른쪽 패널에서 내용·크기·색상·정렬 편집
  • ＋ 도형 — 사각형 도형 추가. 모양(원·삼각형·별 등)·색상·크기 변경 가능

3 컷편집 — 이동 · 트림 · 분할 · 삭제

클립 이동

타임라인에서 클립을 드래그하면 타임라인 상의 시작 위치가 바뀝니다.

트림 (앞/뒤 자르기)

클립의 왼쪽 끝 핸들을 드래그 → 앞부분 잘라내기 (in-point 이동)
클립의 오른쪽 끝 핸들을 드래그 → 뒷부분 잘라내기
영상 클립은 소스 in-point가 자동 보정되어 끊김 없이 재생됩니다.

✂ 분할 (Split)

  • 자르고 싶은 클립을 클릭하여 선택
  • 재생 헤드(노란 세로선)를 원하는 위치로 이동
  • 상단 "✂ 분할" 버튼 또는 키보드 S 누르기
  • 클립이 두 개로 나뉩니다. 영상은 in-point가 자동 계산됩니다.

🗑 삭제

  • 클립 선택 후 상단 "🗑 삭제" 버튼 또는 키보드 Delete / Backspace
  • 오른쪽 패널 → "삭제" 버튼도 동일 기능

4 줌인 / 줌아웃 (Ken Burns 효과)

클립 전체에 걸쳐 서서히 확대/축소되는 카메라 움직임을 추가합니다. 영상·이미지·텍스트 모두 적용 가능합니다.

타임라인에서 클립 클릭하여 선택 오른쪽 패널이 해당 클립의 편집 모드로 전환됩니다
오른쪽 패널 → "애니메이션" 섹션 → 드롭다운에서 "줌 인" 또는 "줌 아웃" 선택 → "추가" 줌 인: 시작 배율(1.0) → 끝 배율(1.4~4.0) 동안 서서히 확대
파라미터 슬라이더로 시작·끝 배율 조정 "시작 배율 · 1.00" "끝 배율 · 1.40" 슬라이더를 드래그하면 미리보기가 실시간 반영됩니다
💡 빠른 방법 (영역 선택 → 줌인 자동 적용)
영역 선택 도구로 드래그 후 "영역으로 줌" 버튼을 누르면 영역 크기에 맞는 Ken Burns 줌인 애니메이션이 자동으로 추가됩니다. 아래 5번 설명 참조.

③ 텍스트 명령으로 적용

영역 지정 후 텍스트 입력란에 키워드를 입력하고 Enter 또는 "명령 적용" 버튼을 누릅니다.

  • 줌 또는 확대 → 영역으로 줌인
  • 블러 또는 흐림 → 영역 블러
  • 모자이크 → 모자이크
  • 스포트라이트 또는 어둡 → 스포트라이트
  • 강조 또는 박스 → 강조 박스
  • 흑백 → 영역 흑백
  • 밝 → 영역 밝게
💡 영역으로 줌 — 동작 방식
"영역으로 줌"을 적용하면 두 가지가 동시에 추가됩니다:
① zoomToRegion 효과 — 해당 영역을 화면 전체로 리프레임 (정적)
② Ken Burns 줌인 애니메이션 — 영역 크기 비율에 맞춘 배율로 클립 시작→끝 점점 확대
오른쪽 패널 → 애니메이션 섹션에서 "줌 인 (Ken Burns)"의 시작·끝 배율을 추가로 조정할 수 있습니다.

8 등장 트랜지션

클립 선택 → 오른쪽 패널 → "등장 트랜지션" 드롭다운으로 클립이 화면에 나타날 때 재생되는 효과를 지정합니다.

  • 없음 — 즉시 나타남 (기본값)
  • 페이드 — 서서히 나타남
  • 슬라이드 — 왼쪽에서 밀려 들어옴
  • 와이프 — 오른쪽에서 닦아내듯 나타남
  • 플립 — 3D 뒤집기 효과
  • 세로 와이프 — 위에서 아래로 닦아내듯 나타남

트랜지션 선택 후 "길이(프레임)" 슬라이더로 지속 시간을 조정하세요.

9 키보드 단축키

Space 재생 / 일시정지
S 플레이헤드에서 클립 분할
Delete 선택된 클립 삭제
Backspace 선택된 클립 삭제

텍스트 내용·숫자를 입력 중일 때는 Space·S·Delete 단축키가 작동하지 않습니다. 입력 완료 후 빈 영역을 클릭해 포커스를 해제하세요.
